# Contributing
Contributions are welcome but limited to clearly scoped, documented changes discussed in advance.

## Reporting Bugs
When reporting a bug [open an Issue](https://github.com/MirathXR/MirathXRabat/issues) and include:
- Steps to reproduce.
- Expected / actual behaviour.
- Device / Unity version.
Only reproducible bugs will be reviewed.

## Suggesting Features
When suggesting a [new feature](https://github.com/MirathXR/MirathXRabat/issues), please describe:
- The problem or need.
- Why it fits the project scope.
- Out-of-scope ideas may be declined.

## Code Contributions
Before submitting code:
- Open an Issue to discuss the change.
- Wait for confirmation.
- Submit a Pull Request referencing the Issue.


## Repository Rules
Please follow these recommendations :
- Do not add large binaries (APK, videos, raw scans).
- Keep documentation in /Docs.
- Do not modify core systems without approval.

## Contact
If unsure, open a [Clarification issue](https://github.com/MirathXR/MirathXRabat/issues) or contact the maintainers directly (email in [Authors](https://github.com/MirathXR/MirathXRabat/blob/main/AUTHORS.md)).
